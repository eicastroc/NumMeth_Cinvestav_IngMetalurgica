## Acerca de

Aquí se presentan un conjunto de `notebooks` que se utilizan como material de soporte para el curso de **Métodos Numéricos**, que es parte de los cursos propedéuticos que se deben acreditar como parte del proceso de selección para ingresar al programa de **Maestría en Ciencias en Ingeniería Metalúrgica** en **Cinvsestav Unidad Saltillo**.

El curso, con duración de 30 horas, abarca los siguientes tópicos:

- Ecuaciones de una variable.
- Ecuaciones algebráicas simultaneas.
- Interpolación y aproximación polinómica.
- Integración numérica.
- Ecuaciones diferenciales ordinarias (valor inicial).
- Ecuaciones diferenciales ordinarias (valor a la frontera).
- Ecuaciones diferenciales parciales.

Al final del curso, se realiza una evaluación escrita.

## Instrucciones

### Instalación de python

Descargar python del [sitio oficial](https://www.python.org/).

Durante el proceso de instalación, habilitar la opción que permite extender el tamaño del camino del systema (`PATH`).

### Instalación de IDE

Se puede utilizar el IDE que se desee.

Si se selecciona `vscode`, descargarlo del [sitio oficial](https://code.visualstudio.com/).

Instalar las extensiones de `python` y `jupyter-notebook`.


### Crear un entorno virtual (virtual environment)

Ejecutar el comando que se muestra debajo, en el mismo directorio que será la carpeta de trabajo.

```bash
python -m venv .venv
```

### Instalar librerias

Ejectuar el comando que se muestra debajo, en el mismo directorio que será la carpeta de trabajo.

```bash
pip install -r requirements.txt
```
